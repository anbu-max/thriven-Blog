# 🖋️ Thriven Editorial Guide

Welcome to the editor's manual for **Thriven**. This guide outlines how to create professional, high-impact blog posts using the custom CMS.

## 🚀 How to Create a New Blog

1.  **Access the Console**: Navigate to `/admin` and login with your credentials.
2.  **Navigation**: Use the "Add New Blog" tab.
3.  **Visuals**: Upload a high-resolution thumbnail (4:3 aspect ratio recommended).
4.  **Metadata**:
    *   **Title**: Use a "Power Hook" (e.g., "The Paradox of...", "The Silent Architect...").
    *   **Category**: Select the appropriate domain (Tech, Philosophy, Startup, Anime, Stories).
    *   **Author**: Use your name (Default: Anbu).

## 🎨 Content Styling Guide

Our editor supports a suite of HTML-based styles to keep the blog looking professional and "techy":

### 1. Typography
*   **Headings**: Use the `H3` tag for section breaks.
*   **Body Text**: Standard text should always be wrapped in `<p>` tags for proper spacing.
*   **Italics**: Use `<i>` for emphasis. In our system, this renders as **Bold Indigo** text for maximum impact.

### 2. Layout Elements
*   **Quotes**: Use the `Quote` button to wrap profound statements in a Stoic-style blockquote.
*   **Lists**: Use the `List` button for bullet points.
*   **Media**: Click "Add Images" to insert multiple high-quality photos directly into the article body. These will be automatically styled with rounded corners and shadows.

## 📱 Responsive Content
The system automatically handles responsiveness:
*   **Images** scale according to the screen size.
*   **Text** maintains readability with optimized line heights.
*   **Shadows and Borders** adjust for mobile viewports to prevent clutter.

## 🔄 Admin Persistence
Your session is currently configured to stay active for **7 days**. If you refresh the page or close the browser, you will remain logged in until the session specifically expires or you manually click "Logout".
